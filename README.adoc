=== Lossy translation with a highly contextual language

ilo ante sitelen toki may also be translated as...

> the translator is a sign language

> the translator is printed

> a sign language translator (interpreter)

> a pencil is a sign language

> the translator is a pen

while machine translator might be translated a number of ways, depended on detail, complexity, and existing context...

> ilo pi ante toki (machine of changing language)

> ilo pi toki ante (machine of language modifying)

if we need more detail...

> lawa ilo li toki ante (machine mind is changing language) -> computer translator

> ilo toki ante pi ilo nasin sona (machine language changer of machine knowledge growth) -> machine learning language translator

> ilo toki ante pi toki pona e toki Inli  (machine language changer of good language and 'English' language) -> translator of toki pona and english

> ilo pi nasin sona ilo toki ante pi toki pona en toki Inli (machine <of machine knowledge growth> language changer <of good language and 'English' language>) -> machine learning translator of toki pona and english

Remember, there are no verbs or direct objects here (no li or e), so ALL of the above are *single words*. It makes the analytical aspects of German grammar seem quite minimal!
